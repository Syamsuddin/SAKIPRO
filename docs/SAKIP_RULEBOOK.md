# SAKIPRO v1.0 Review Rulebook

Version: 1.0  
Status: Blueprint Revision  
Purpose: Menjadikan review SAKIP konsisten, dapat diuji, dan selaras dengan Permenpan RB 88/2021.

---

## 1. Komponen dan Bobot Evaluasi

SAKIPRO mengacu pada standar penilaian nasional:

| Komponen | Bobot | Fokus Utama |
|---|---|---|
| Perencanaan Kinerja | 30% | Kualitas dokumen (Renstra, Renja, PK), Cascading, Pohon Kinerja |
| Pengukuran Kinerja | 30% | Validitas data, monitoring rutin, dashboard, penggunaan data |
| Pelaporan Kinerja | 15% | Kualitas LKjIP, analisis keberhasilan/kegagalan, efisiensi |
| Evaluasi Internal | 25% | Kualitas reviu APIP (Inspektorat), tindak lanjut LHE |

---

## 2. Prinsip Penilaian (Desk Evaluation)

SAKIPRO mensimulasikan peran Evaluator dalam tahap Desk Evaluation dengan fokus:

1. **Outcome-Oriented:** Membedakan indikator hasil (outcome) vs aktivitas/output admin.
2. **Logical Framework:** Memastikan Sasaran -> Program -> Kegiatan -> Anggaran memiliki rantai logika yang tidak putus.
3. **Analyticity:** Menilai apakah Pelaporan (LKjIP) bersifat analitis (menjawab "mengapa") bukan sekadar naratif.
4. **Data Consistency:** Cross-check data antara dokumen perencanaan, perjanjian, dan laporan.

## 3. Rubrik Indikator Kinerja (Refined)

| Rule ID | Kriteria | Kondisi Tidak Memenuhi (Critical Fail) |
|---|---|---|
| IND-001 | Berorientasi Outcome | Hanya mengukur jumlah orang, jumlah dokumen, atau frekuensi rapat |
| IND-002 | Definisi & Batasan Jelas | Rumusan multitafsir atau tidak menjelaskan cakupan pengukuran |
| IND-003 | Formula Tersedia | Metode hitung (numerator/denominator) tidak ditemukan |
| IND-004 | Satuan Ukur Standar | Satuan tidak relevan (misal: "Kegiatan" untuk indikator outcome) |
| IND-005 | Sumber Data Valid | Tidak menyebutkan aplikasi/dokumen asal data atau unit penanggung jawab |
| IND-006 | Target Ambisius & Konsisten | Target tidak sinkron antar dokumen atau menurun tanpa alasan logis |
| IND-007 | Baseline Tersedia | Tidak ada data awal (baseline) sebagai pembanding untuk mengukur kemajuan |
| IND-008 | Target Realistis vs Baseline | Target terlalu mudah (di bawah capaian tahun lalu) atau tidak ada kenaikan bermakna dari baseline; atau loncat drastis tanpa justifikasi anggaran/program |
| IND-009 | Sensitivitas & Validitas Proxy | Indikator tidak benar-benar mengukur apa yang diklaim — misal: "jumlah dokumen yang diterbitkan" dipakai sebagai proxy "peningkatan tata kelola" tanpa rantai kausal yang jelas |

---

## 4. Rubrik Perjanjian Kinerja (PK)

| Rule ID | Kriteria | Kondisi Tidak Memenuhi |
|---|---|---|
| PK-001 | Selaras Perencanaan | Sasaran/Indikator PK tidak ditemukan di Renstra/Renja/IKU |
| PK-002 | Konsistensi Nama, Satuan, Definisi | Nama indikator, satuan, atau definisi berbeda antara PK dan IKU/Renstra/Renja |
| PK-003 | Kesesuaian Target | Angka target berbeda dari IKU/Renstra tanpa catatan revisi resmi |
| PK-004 | Dukungan Program Nyata | Tidak ada program/kegiatan yang secara logis mendukung sasaran PK |
| PK-005 | Penanggung Jawab Jelas | Unit pelaksana atau pejabat penanggung jawab sasaran PK tidak disebutkan |
| PK-006 | Milestone Waktu Tersedia | Tidak ada target triwulanan atau semesteran; hanya target akhir tahun — evaluator modern mensyaratkan rencana aksi berbasis milestone |

---

## 5. Rubrik LKjIP (Analisis & Akuntabilitas)

| Rule ID | Kriteria | Kondisi Tidak Memenuhi |
|---|---|---|
| LKJ-001 | Perbandingan Realisasi | Menyajikan realisasi tanpa membandingkan dengan target PK |
| LKJ-002 | Analisis Penyebab (Deep) | Hanya narasi "tercapai", tidak menjelaskan faktor kunci/hambatan |
| LKJ-003 | Efisiensi Anggaran | Tidak membandingkan % Capaian Kinerja vs % Serapan Anggaran |
| LKJ-004 | Tindak Lanjut Nyata | Rekomendasi LHE tahun lalu tidak dijawab dengan bukti perbaikan |
| LKJ-005 | Evidence Support | Klaim prestasi besar tanpa lampiran bukti/sumber dokumen valid |
| LKJ-006 | Tren Capaian Multi-Tahun | Tidak ada perbandingan capaian minimal 3 tahun terakhir; evaluator menilai konsistensi dan arah perubahan kinerja |
| LKJ-007 | Kedalaman Analisis Narasi | Narasi hanya melaporkan angka tanpa menjelaskan "mengapa bisa tercapai", "apa yang menghambat", dan "apa yang akan berbeda tahun depan" |

---

## 6. Rubrik Cascading (Rantai Logika Kinerja)

| Rule ID | Kriteria | Kondisi Tidak Memenuhi |
|---|---|---|
| CAS-001 | Rantai Logika Utuh | Rantai Sasaran → Indikator → Program → Kegiatan → Subkegiatan terputus atau ada komponen yang tidak terhubung |
| CAS-002 | Relevansi Kegiatan | Kegiatan/subkegiatan tidak relevan atau tidak mendukung pencapaian sasaran yang dihubungkan |
| CAS-003 | Keselarasan Vertikal RPJMD/RKPD | Sasaran OPD tidak terhubung ke sasaran Pemda di RPJMD atau RKPD; OPD bekerja di koridor yang berbeda dari arah kebijakan daerah |

---

## 7. Rubrik Evidence (Bukti Dukung)

| Rule ID | Kriteria | Kondisi Tidak Memenuhi |
|---|---|---|
| EVD-001 | Relevansi Evidence | Evidence yang dilampirkan tidak relevan dengan klaim kinerja yang didukung |
| EVD-002 | Verifiabilitas Evidence | Evidence tidak dapat diverifikasi (hanya self-report tanpa data pendukung eksternal) |
| EVD-003 | Kecukupan Evidence | Jumlah atau cakupan evidence tidak memadai untuk mendukung klaim kinerja |
| EVD-004 | Ketepatan Waktu Evidence | Tanggal evidence berada di luar periode klaim kinerja; evidence retroaktif atau dari tahun berbeda tidak dapat memvalidasi klaim periode berjalan |

## 8. Predikat Penilaian (Simulation)

SAKIPRO dapat memberikan estimasi predikat berdasarkan skor kumulatif:
- **AA (>90):** Sangat Memuaskan (Efisien, Efektif, Berdampak).
- **A (>80-90):** Memuaskan.
- **BB (>70-80):** Sangat Baik.
- **B (>60-70):** Baik.
- **CC (>50-60):** Cukup (Hanya administratif).
- **C (>30-50):** Kurang.
- **D (0-30):** Sangat Kurang.
